# Ark
### Curated development environment for web and deep learning.


## How to use
Docker images published to DockerHub and Github Container Registry(ghcr.io). 

### Docker Hub
* base,latest
  kevgao/ark:base
  kevgao/ark:latest
* web
  kevgao/ark:web
* deeplearning
  kevgao/ark:deeplearning

### Github Container Registry
* base, latest
  ghcr.io/ark:base
  ghcr.io/ark:latest
* web
* deeplearning
